# Gemini Project Instructions

This project contains the source code for the TSGabrielle home page.

## Project Structure

The project is organized into subdirectories, each representing a different page or component of the website.

- `tsgabrielle®_admin_dashboard_overview/`: Admin dashboard overview page.
- `tsgabrielle®_admin_product_management/`: Admin product management page.
- `tsgabrielle®_contact_and_information_page/`: Contact and information page.
- `tsgabrielle®_faq_and_reviews_page/`: FAQ and reviews page.
- `tsgabrielle®_legal_and_sitemap_page/`: Legal and sitemap page.
- `tsgabrielle®_mobile_3d_glass_checkout/`: Mobile 3D glass checkout page.
- `tsgabrielle®_mobile_thank_you_celebration/`: Mobile thank you celebration page.
- `tsgabrielle®_the_brand_page/`: The brand page.

Each directory contains:

- `code.html`: The HTML code for the page.
- `screen.png`: a screenshot of the page design.

## How to Work with This Project

When asked to make changes, refer to the `code.html` to understand the structure and content of the page. The `screen.png` can be used as a visual reference for the desired appearance.
